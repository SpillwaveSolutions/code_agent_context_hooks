# Command Explanation Required

Before executing this command, please provide:

## What
A plain English explanation of what this command does.

## Why
The reason this command is needed for the current task.

## Risk Assessment
- **Reversible**: Can this action be undone?
- **Side Effects**: What other systems or files might be affected?
- **Data Loss**: Is there any risk of data loss?

## Alternatives Considered
What other approaches were considered and why this one was chosen.
